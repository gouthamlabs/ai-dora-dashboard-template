# AI & DORA Metrics Dashboard Template

This repo provides a ready-to-fork template for tracking DORA metrics, enforcing AI policy, and managing AI pilot intake in regulated environments (e.g., pharma/LSH).

## Features
- DORA metrics collection and export
- AI usage policy enforcement
- AI pilot proposal intake template
- Sample dashboard integration
- Compliance and Responsible AI workflows

## Getting Started
1. Fork this repo.
2. Update `POLICY/AI_USAGE_POLICY.md` for your organization.
3. Connect your CI/CD data sources.
4. Customize workflows and templates as needed.
