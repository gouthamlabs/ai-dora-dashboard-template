# AI Usage Policy

**Scope:** All code, data, and workflows in this repository.
**Allowed Tools:** GitHub Copilot, approved LLMs, internal agents.
**Data Guardrails:** No patient/PII data in prompts. Use only validated internal sources.
**Responsible AI:** Outputs must be verified. Document decisions for clinical/regulated changes.
**Governance:** Violations auto-flagged. Monthly audits. Policy updates via governance board.

Contributors must acknowledge this policy in every PR.
